Installation
------------
.. note::

   This install method will change soon, as I will be moving to the unity package manager soon.
   
Installing all of Unity Utils
~~~~~~~~~~~~~~~~~~~~
1.) To install Unity Utils , first go to the `GitHub <https://github.com/my-dimons/Unity-Utils/tree/main>`_.

2.) Click the green button and download the project as a .zip.

3.) Unzip the downloaded file

3.) Finally load the folder "Unity Utils" inside the extracted file into Unity Hub and launch!

Downloading Script Utils
~~~~~~~~~~~~~~~~~~~~~~~~

1.) Find the specific scripts utils you want to import into your game (Find Script Utils `here <https://github.com/my-dimons/Unity-Utils/tree/main/Unity%20Utils/Assets/Script%20Utils>`_).

3.) Download these scripts, then drag them into your unity project
