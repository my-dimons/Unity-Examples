UIButtonSfx
==========

**NAMESPACE:**
   `UnityUtils.ScriptUtils.UI`
   
Usage
-------------
Add a 'UIButtonSfx' Script to a UI button and add a hover enter, hover exit, and click sound effect. 
You don't need to add all of these sound effects, but if you don't add all of them you can enable 'emptySfxErrorMessages'. 
You can also edit the volume for each type of sound effect and set the volume type.
